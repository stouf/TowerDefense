Peut être utilisé pour la création d'un nouvel ennemi, ou pour le chargement d'un ennemi lors d'une sauvegarde de la partie.

Parameters:
Position: position de l'ennemi.
TypeDeplacement: type de déplacement de l'ennemi.
taille: taille de l'ennemi, en unités.
resistance: resistance de l'ennemi.
vitaliteBase: vitalité de base de l'ennemi.
vitalite: vitalité courante de l'ennemi.
vitesse: vitesse de base de l'ennemi.
timerMalusVitesse: nombre de tick de timer restant avant la fin du malus de vitesse suite à une touche par une bille de Paint-Ball. 0 si pas de malus.

void TOWERDEFENSE::Ennemi::recevoirProjectile ( Projectile *  projectile) [virtual]

Effectue les opérations de base lorsque l'ennemi est atteint par un projectile :

  • Inflige les dégats via la méthode recevoirDegats()
  • Suivant les projectiles, effectue différentes opérations (ralentissement si bille de Paint-Ball, dégats aux voisins si boule de pétanque).
    Parameters:
    Projectile: projectile qui vient d'atteindre la cible

Reimplemented in TOWERDEFENSE::Fourmi, and TOWERDEFENSE::Moustique.
